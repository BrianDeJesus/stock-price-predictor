# Stock Predictor
This program predicts near future closing stock prices and plots them on a graph

# Command Line Argument
 * The company's symbol/abbreviation (AAPL, TSLA, GOOGL, etc.)
 * Not case sensitive

 **Example**
 ```
$ python3 stock_predictor.py aapl
 ```
